# Student Grading System

## Description
This is a simple Student Grading System built using Django and Django Templates.
The system allows management of students, subjects, and marks, and automatically
assigns grades based on the marks entered.

This project was developed for learning and academic purposes.

## Features
- Add students
- Add subjects
- Enter marks
- Automatically calculate grades
- Display results using Django templates

## Technologies Used
- Python
- Django
- Django Templates
- SQLite (default Django database)

## Grading Logic
- 75 – 100 : A
- 60 – 74  : B
- 50 – 59  : C
- Below 50 : F
